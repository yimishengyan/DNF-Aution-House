## 🚀 技术栈

### 后端技术栈 (`DNF/`以及`dnf_sql`)
| 技术 | 说明 |
|------|------|
| ​**Spring Boot**​ | 核心框架 |
| ​**MyBatis**​ | ORM框架 |
| ​**Lombok**​ | 代码简化工具 |
| ​**MySQL 8.0+​**​ | 数据存储 |
| ​**Maven**​ | 项目管理 |

### 前端技术栈 (`frontEnd/`)
| 技术 | 说明 |
|------|------|
| ​**Vue 2.x**​ | 核心框架 |
| ​**Vue Router**​ | 页面路由 |
| ​**Vuex**​ | 状态管理 |
| ​**Element UI**​ | UI组件库 |
| ​**Axios**​ | HTTP客户端 |
| ​**Node.js v14.21.3**​ | 运行环境 |
